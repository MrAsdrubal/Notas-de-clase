Git es  el sistema de control de versiones moderno más utilizado del mundo. Git es un proyecto de código abierto maduro y con un mantenimiento activo que desarrolló originalmente Linus Torvalds, el famoso creador del kernel del sistema operativo Linux, en 2005.

### **Definición:**
---

La algoritmia consiste en la búsqueda de soluciones a un problema concreto. 

De manera mas formal se lo podria definir al algoritmo como la transcripción exacta de una serie de instrucciones concretas con las cuales se encuentra una solución.

 Para llegar a la solución de un problema existen distintos pasos y procesos, algunos pueden resultar mas largos otros más cortos y óptimos.

 En conclusión,existe una infinidad de algoritmos para la solución de un determinado problema, si ponemos a resolver a dos personas un mismo problema en lenguaje de programación será obvio que las dos personas lo resolveran de maneras distintas.

};

## ***Directivas del precompilador***

>#include

 Incluye el fichero, cuyo nombre se indica, para su compilación
con el resto del Código
- Si el nombre del fichero va entre <... > se busca en el
directorio include del sistema (/usr/include). 

Por ejemplo
#include <stdio.h>
hace que el fichero /usr/include/stdio.h se incluya en ese
punto para ser compilado con nuestro fichero fuente
- El nombre se puede poner entre comillas ("") y el sistema lo
busca en el directorio actual (tambiénn se puede poner una
ruta absoluta)
- Los ficheros se suelen denominar .h y contienen declaraciones
de tipos, variables y prototipos de funciones (son ficheros
fuente que pueden verse)
#define Define un símbolo, es decir cada ocurrencia de ese s+ímbolo es
sustituida por su definición. Por ejemplo
- El C distingue entre mayúsculas y minúsculas, aunque la
costumbre es utilizar mayúsculas para los símbolos definidos
por el preprocesador

- **Ejemplo**

#define MAX 1024
hace que todas las ocurrencias de la cadena MAX dentro del
código son sustituidas por 1024
* El código que se compila lleva 1024 donde iba MAX
* Si MAX iba dentro de comill as ''...'', no se sustituye
